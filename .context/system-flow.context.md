# Fluxo Principal do Sistema

## Tela Estudos
Responsável por:
- importar editais
- importar PDFs
- importar planos
- dashboard
- pomodoro
- quiz
- revisões
- anki

## Fluxo
1. usuário importa edital
2. IA detecta cargos
3. usuário escolhe cargo
4. IA gera cronograma
5. dashboard atualiza
6. pomodoro usa agenda

## Regras
- quiz gera revisão automática
- erros criam flashcards
- revisões entram na agenda
